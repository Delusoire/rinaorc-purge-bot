```
rinaorc-purge-bot is a mineflayer command-driven bot specialised to help rinaorc players speed up the leveling up process in the ~unique~ gamemode: The Purge.

To setup this bot, first install <git> and <nodejs> then execute the following commands on the terminal:
# git clone 'https://github.com/Gh-raf/rinaorc-purge-bot'
# cd rinaorc-purge-bot
# npm install

To kickstart the AI's cognitive subsystem, all you need is to run:
# node start
```
